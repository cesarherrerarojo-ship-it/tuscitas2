# Pasos en Firebase Console

1. App Check: habilitar proveedores y modo “Enforced” para prod.
2. reCAPTCHA Enterprise: crear y asociar Site Key al dominio de producción.
3. Firestore Rules: pegar contenido de `firestore.rules` y publicar.
4. Storage Rules: pegar contenido de `storage.rules` y publicar.
5. Authentication: revisar proveedores y custom claims según necesidad.
