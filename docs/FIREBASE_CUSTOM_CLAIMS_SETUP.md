# Configuración de Custom Claims

Referencias:
- Script existente: `scripts/setAdminClaim.js`.

Ejemplo de ejecución:
- Establecer `admin: true` a un usuario por UID usando el script.
- Verificar en cliente con `getIdTokenResult()` para leer `claims`.

Claims usuales:
- `admin`, `concierge`, `vipEligible`.
